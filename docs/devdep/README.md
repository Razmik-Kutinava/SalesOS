## Как пользоваться

1. Прочитай `01-obzor-pravila-styki.md` — общая рамка и таблица стыков.
2. Углубляйся в файлы ролей, которые участвуют в твоём треке.
3. При изменении процесса обновляй соответствующий файл и при необходимости таблицу в `01`.
